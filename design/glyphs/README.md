# NΞBU Canonical System Glyph

This glyph set defines the canonical **Ξ primitive** for NΞBU.

## Files

- `nebu-xi-base.svg`: flat, no glow, white lines.
- `nebu-xi-active.svg`: subtle signal-green glow for active state.
- `nebu-xi-signal.svg`: controlled stronger signal-green glow for signal state.

## Geometry Contract

- Exactly three horizontal parallel lines.
- Line thickness: `1.50` units for all three lines.
- Equal vertical spacing via fixed y positions: `4.00`, `7.25`, `10.50`.
- Shared x alignment and width: `x=3`, `width=10`.
- Square edges (`rect` with no corner radius).

## Visual Contract

- Background: `#05070A`.
- Base foreground: `#FFFFFF`.
- Active/Signal foreground: `#00FF41`.
- Flat, ultra-minimal rendering.
- No added symbol geometry beyond the three-line Ξ form.
